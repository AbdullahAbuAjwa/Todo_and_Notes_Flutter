# Todo and Notes - Flutter

Todo and Notes app.


## Advantages of the application:
- Add notes, you can edit, favorite and share them.
- Add a to-do list, and choose the appropriate color for the card.
- Multilingual "Arabic and English".
- Supported dark mode.
- Responsive.
